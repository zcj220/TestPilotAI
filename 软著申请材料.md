## 十一、软件的主要功能（500～1300 字）

测试领航员 AI 自动化测试软件（TestPilot AI）是一款面向软件开发者和测试工程师的人工智能驱动自动化测试平台，主要提供以下核心功能：

**1. 多平台自动化测试执行**  
支持 Web 浏览器应用、iOS 原生应用、Android/Flutter 原生应用、微信小程序及 Windows 桌面应用五大平台的自动化测试。测试引擎基于 Playwright（Web）、Appium（iOS/Android）、微信开发者工具自动化接口（小程序）及 pywinauto（桌面）实现跨平台统一驱动，用户通过统一的蓝本文件（JSON 格式测试脚本）描述测试场景，引擎自动完成操作执行。

**2. 蓝本模式智能测试**  
用户通过 JSON 格式蓝本文件定义测试页面、元素选择器、测试场景及预期结果，引擎解析蓝本后自动执行导航、点击、输入、选择、截图、断言等操作动作，每步执行结果与预期对比，自动生成通过/失败报告。蓝本文件支持按功能模块拆分独立维护，适合持续集成场景。

**3. AI 视觉断言与截图分析**  
集成豆包视觉多模态大模型（Doubao-Seed-1.8），当断言步骤难以通过 DOM 节点定位时，自动截取当前屏幕并发送给 AI 进行视觉分析，判断预期文本或元素是否满足条件，实现"所见即所测"的语义级断言能力，大幅减少选择器维护工作量。

**4. AI 修复闭环（自愈机制）**  
测试发现 Bug 后，系统自动将失败步骤信息、截图及预期结果整理为结构化 Bug 报告，通过 MCP（Model Context Protocol）协议回传给 Cursor/Windsurf 等 AI 编程工具，触发自动代码修复，修复完成后重新执行测试，形成测试→修复→重测的完整自愈闭环。

**5. IDE 插件集成**  
提供 Visual Studio Code、Windsurf、Cursor 三平台插件（VSIX 格式），通过侧边栏面板实现一键启动引擎、选择蓝本、观看实时测试日志、查看测试报告、分享修复经验等操作，将自动化测试能力无缝嵌入开发工作流，无需切换工具。

**6. 社区经验库**  
内置社区经验库，开发者在测试后遇到并修复的 Bug 可匿名化共享至社区，其他用户搜索相似错误时系统智能推荐相关修复经验，形成集体知识积累。平台提供经验搜索、点赞采纳、贡献者排行榜等社交功能，并配套 React 技术栈开发的 Web 门户网站供用户访问。

**7. MCP 协议支持**  
实现 Model Context Protocol 标准接口，允许 Cursor、Windsurf 等 AI 编程工具通过自然语言指令直接调用测试引擎，发起测试任务、获取测试报告并触发代码修复，实现 AI 原生的开发测试协作模式。

**8. 本地记忆与测试历史**  
测试历史、经验教训及已识别的 Bug 模式持久化存储于本地 SQLite 数据库，引擎根据历史数据智能调整测试策略，避免重复踩坑，持续提升测试精准度与修复效率。

### 生成步骤（PowerShell）

```powershell
# 拼接全量代码
$files = @("main.py","cli.py","src/auth/service.py","src/auth/routes.py",
           "src/core/ai_client.py","src/api/routes.py",
           "extension/src/extension.ts","extension/src/sidebarProvider.ts",
           "web/src/App.jsx","web/src/pages/LoginPage.jsx","web/src/pages/HomePage.jsx")
$all = $files | ForEach-Object { Get-Content $_ }
$all | Set-Content 软著代码全量.txt

# 取前30页（前1500行）
$all | Select-Object -First 1500 | Set-Content 软著代码_前30页.txt

# 取后30页（后1500行）
$all | Select-Object -Last 1500 | Set-Content 软著代码_后30页.txt
```
